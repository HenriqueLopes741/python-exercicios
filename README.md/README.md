# Python Exercícios

Repositório criado para armazenar exercícios e desafios em Python, organizados por temas.

##  Conteúdos abordados
- Variáveis e tipos de dados
- Condições (`if`, `elif`, `else`)
- Laços de repetição (`for`, `while`)
- Listas, tuplas e dicionários
- Funções
- Desafios práticos de lógica

## Objetivo
Praticar Python diariamente, reforçar a lógica de programação e criar um histórico de evolução no GitHub.

## Tecnologias
- Python 3.x

##  Observação
Os exercícios foram feitos durante os estudos em programação e podem ser refatorados conforme a evolução.
